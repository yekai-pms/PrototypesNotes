## 通用需求功能描述：
### 名词说明
---
1. 导入方式切换区域
   1. 批量录入：指通过导入模板批量导入多道试题内容。
   2. 单题录入：指通过填写单个题目信息来导入单道试题内容。
2. 导入内容区域（单题录入）
   1. 试题分类：指试题题干中的文字内容。
   2. 试题类型：指试题属性「单选题/多选题/判断题/填空题」四种类型中的一种。
   3. 试题难度：指试题属性「简单/一般/困难」三种类型中的一种。
   4. 重置：指重置用户页面为初始页面。
   5. 录入：指将单题题目导入到数据库中。点击后，检查导入内容是否合规，并弹出提示（具体内容见交互）。
   6. 题干：
      1. 指试题的题干内容。
      2. 无字符数量限制，可以输入中文、英文、数字、特殊符号。
   7. 选项：
      1. 指试题的答案选项，根据「单选题/多选题/判断题/填空题」类型切换显示个数（具体内容详见单题录入）。
      2. 无字符数量限制，可以输入中文、英文、数字、特殊符号。
   8. 解析：
      1. 指试题内容的答案解析。
      2. 无字符数量限制，可以输入中文、英文、数字、特殊符号。
3. 导入内容区域（批量录入）
   1. 温馨提示：指“使用批量导入功能导入的试题将存放在导入模板编辑的分类中。”提示内容。
   2. 重置：指重置用户页面为初始页面。
   3. 录入：指将用户上传批量录入模板文件导入到数据库中。点击后，检查导入内容是否合规，并弹出提示（具体内容见交互）。
   4. 选择文件：指用户选择电脑本地文件进行上传。
   5. 下载Excel导题模板：指用户下载导题规范模板。包含自适应试题分类选择，由后端提供。
   6. 模板使用说明：指“模板导入的步骤。”说明文字。


## 功能需求点说明
### 需求点1：添加试题初始状态
1. 用户点击 "垂直导航菜单-试题管理-添加试题" 按钮进入添加试题页面。
2. 导入方式切换区域
   1. 默认显示状态为「单题录入」Tab 页面。
   2. 显示状态下的「单题录入、批量录入」高亮显示。
3. 导入内容区域（单题录入）
   1. 试题分类：
      1. 必选项，显示内容为空。用户点击后进行树形结构分类的选择。可编辑。
   2. 试题类型：
      1. 默认选择为单选题。
      2. 根据选择试题类型，显示「选项」发生变化。
   3. 试题难度：默认选择为简单。
   4. 题干：
      1. 默认显示提示文本内容，“题干，点此编辑（必填）”。
      2. 默认显示3行文本高度。
   5. 选项：
      1. 默认显示提示文本内容，“选项，点此编辑；选中即设置为正确答案（必填）。”
      2. 默认显示1行文本高度。
      3. 默认选择第一行选项为正确答案。
         1. 「选项-删除」：删除当前选项。删除时提示是否删除。
         2. 「选项-上移」：首行选项点击无效，非首行选项，将当前选项向上移动一项。
         3. 「选项-下移」：末行选项点击无效，非首行选项，将当前选项向下移动一项。
   6. 解析：
      1. 默认显示提示文本内容，“题目解析，点此编辑（选填）。”
4. 导入内容区域（批量录入）初始状态。

### 需求点2：切换「试题类型」
1. 导入内容区域（单题录入）
   1. 试题类型：默认选择为单选题。根据选择试题类型，显示「选项」发生变化。
   2. 根据「单选题/多选题/判断题/填空题」类型切换显示个数。
      1. 添加选项：指添加选项的个数，至多添加至8项。
   3. 选项显示内容详细说明：
      1. 「单选题题型」：
         1. 默认显示4个选项，允许添加选项至第8个，添加选项至第八项时，隐藏「添加选项」按钮。
         2. 默认勾选第一选择为正确答案，用户可选择任意唯一值为正确答案。
      2. 「多选题题型」：
         1. 默认显示4个选项，允许添加选项至第8个，添加选项至第八项时，隐藏「添加选项」按钮。
         2. 默认勾选「第一行选项」为正确答案，用户可选择任意多项值为正确答案。
      3. 「判断题题型」：
         1. 默认显示2个选项，无法添加选项，隐藏「添加选项」按钮。
         2. 默认勾选「第一行选项」为正确答案。
         3. 选项内容显示为，不可更改：
            1. 第一行选项：「A：正确」
            2. 第二行选项：「B：错误」
      4. 「填空题题型」：
         1. 默认显示1个选项，允许添加选项至第8个，添加选项至第八项时，隐藏「添加选项」按钮。
            1. 「范例」-->>选项A:默认显示提示文本内容，"参考答案，点此编辑，回车保存;支持保存多个答案(必填)".

### 需求点3：「题干、选项、解析」默认状态与输入内容
1. 「题干」
   1. 此项为「必填项」
   2. 文本框  默认显示提示文本内容，“题干，点此编辑（必填）”
   3. 文本框  默认显示 3 行文本高度。
   4. 交互内容说明
      1. 点击提示文本区域，提示文本内容消失，显示输入光标。
      2. 用户鼠标点击其他区域，取消输入。若无输入内容。重新显示提示文本。
      3. 显示文本框高度跟随输入文本数量变大。
2. 「选项-内容」
    1. 此项为「必填项」
    2. 文本框  默认显示提示文本内容，“选项，点此编辑；选中即设置为正确答案（必填）”
    3. 文本框  默认显示 1 行文本高度。
    4. 文本框  在选项末端显示「删除、上移、下移」按钮
       1. 「选项-删除」：删除当前选项。删除时提示是否删除。
       2. 「选项-上移」：选项向上移动按钮。
       3. 「选项-下移」：选项向下移动按钮。
    5.  交互内容说明
        1.  点击文本框，提示文本内容消失，显示输入光标。
        2.  用户鼠标点击非文本框区域，取消输入。若无输入内容。重新显示提示文本。
        3.  显示文本框高度跟随输入文本数量变大。
        4.  在选项末端显示「删除、上移、下移」按钮
            1.  「选项-删除」：删除当前选项。删除时提示是/否删除。-->>弹出提示框。
            2.  「选项-上移」：首行选项点击无效，非首行选项，将当前选项向上移动一项。
            3.  「选项-下移」：末行选项点击无效，非首行选项，将当前选项向下移动一项。
        5.  「特殊」此项仅针对「填空题」题型：
            1.  输入内容后“↩︎”后保存为选项内容，并列显示多个文本块，无先后排序性。
            2.  点击文本块右上角的❌可以将文本块删除。
3. 「选项-选项按钮」
   1. 此项为「必填项」
   2. 在选项内容前显示选项按钮。
   3. 选项按钮后附带选项的名称。
      1. 「单选题」：「A、B、C、D、E、F、G、H」至多8项，根据选项内容个数依次显示。
      2. 「多选题」：「A、B、C、D、E、F、G、H」至多8项，根据选项内容个数依次显示。
      3. 「判断题」：「A、B」有且仅有两项。
      4. 「填空题」：无选项按钮；仅按填空数量显示序号。「1、2、3、4、5、6、7、8」至多8项，根据选项内容个数依次显示。
   4. 默认选择「第一行选项」为正确答案。
      1. 「单选题」仅选择一项。
      2. 「判断题」仅选择一项。
      3. 「多选题」可选择多项。
      4. 「填空题」以输入内容为评判正确与否。
4. 「解析」
   1.  此项为「选填项」
   2.  文本框  默认显示提示文本内容，“题目解析，点此编辑（选填）。”
   3.  文本框  默认显示 3 行文本高度。
   4.  交互内容说明
       1.  点击文本框，提示文本内容消失，显示输入光标。
           1.  用户鼠标点击非文本框区域，取消输入。若无输入内容。重新显示提示文本。
       2.  显示文本框高度跟随输入文本数量变大。



### 需求点4：「批量录入」功能

1. 温馨提示：指“使用批量导入功能导入的试题将存放在导入模板编辑的分类中。”提示内容。
2. 重置：指重置用户页面为初始页面。
3. 录入：指将用户上传批量录入模板文件导入到数据库中。点击后，检查导入内容是否合规，并弹出提示（具体内容见交互）。
4. 文件名称：指用户上传使用的文件名称。
5. 选择文件：指用户选择电脑本地文件进行上传。
6. 下载Excel导题模板：指用户下载导题规范模板。包含自适应试题分类选择，由后端提供。
7. 模板使用说明：指“模板导入的步骤。”说明文字。
8. 交互内容说明
   1. 选择文件：
      1. 点击后弹出本地文件路径供用户进行选择文件。
      2. 用户上传文件后，「文件名称」显示用户上传文件。
   2. 下载Excel导题模板：
      1. 点击后下载系统预设导题模板。
      2. 导题模板包含项目有
         1. 试题内容：「必填」
         2. 试题类型：「必填」，用下拉框勾选，不支持自定义。
         3. 试题选项：「必填」,「A、B、C、D、E、F、G、H」至多8项，根据题型判断是否填写正确。
            1. 「判断题」仅支持两项选项
            2. 「单选题、多选题、填空题」支持至多8项内容。
         4. 试题难度：「必填」，用下拉框勾选，不支持自定义。
         5. 正确答案：「必填」，「A、B、C、D、E、F、G、H」至多8项，根据题型判断是否填写正确。
            1. 「判断题、单选题」仅支持一项选项为正确答案。
            2. 「多选题、填空题」支持至多8项内容为正确答案。
         6. 答案解析：
            1. 「选填项」
            2. 无字符数量限制，可以输入中文、英文、数字、特殊符号。
